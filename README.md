# Promptmaker Alfred Workflow

一个定制的 Alfred Workflow，用于生成基于给定参数的艺术提示。该 Workflow 使用[promptmaker](https://github.com/zeke/promptmaker)库，根据提供的参数生成文本提示。

## 功能介绍

- 根据提供的参数生成艺术提示
- 将生成的提示复制到剪贴板
